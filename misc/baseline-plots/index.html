Employment rates in EU over time - baseline plots

This page provides visualizations for several (un)employment rates in European countries over time, both at the country and regional (NUTS) levels. The data are also presented in searchable tables, and can be downloaded as csv files.

Visualization & Options: The graphs below show how the value of the selected rate, e.g., unemployment, (y-axis) changes over time (x-time). All values are percentages, e.g., a value of 80 at the y-axis means 80%.

  • Different lines correspond to different countries or regions (hovering the mouse over a line shows the country/region name). One or more countries/regions can be selected.
  • By default, the values are normalized with respect to a value of a reference year. For example, if 2008 is selected as the reference year, and the employment rates at 2008 and 2009 are 50% and 40%, respectively, then the normalized value appearing in the graph for 2009 will be 40/50 = 0.8 = 80%. Different reference years can be selected, or normalization can be deselected.
